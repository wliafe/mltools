## 介绍

mltools：一个专注于机器学习工具开发的开源项目。

## 安装命令

推荐使用 uv 安装。

```bash
uv add wliafe-mltools
```

或者使用 pip 安装。

```bash
pip install wliafe-mltools
```

## 使用说明

1. mltools 需要 python3.8 及以上版本。
2. 样例可以参考 example 文件夹。
